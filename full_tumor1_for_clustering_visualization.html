Notes for Following the Scanpy Clustering Tutorial¶

Data loading¶

Import the data I give you like this:

adata = sc.read_10x_h5('filtered_feature_bc_matrix.h5') adata.var_names_make_unique()

Don't use the 10X matrix files in tutorial line # 5

Data processing¶

  1. Use Leiden not Louvain
  2. Beware that dataset-specific examples won't work, like:

sc.pl.umap(adata, color=['CST3', 'NKG7', 'PPBP'])

  1. Mouse genes will usually start with a Capital letter and the rest are lowercase. Or all are lowercase, like in the case of mitochondrial genes:

adata.var['mt'] = adata.var_names.str.startswith('mt-')
